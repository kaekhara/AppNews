# 📰 AppNews — iOS News Reader

A clean iOS app for browsing the latest news, built with Swift and SwiftUI as a final project at **nFactorial iOS Course (2024)**.

---

## 📱 Features

- Browse a curated feed of news articles
- Clean, minimalist card-based UI
- Built entirely with SwiftUI
- Smooth navigation between list and detail views

---

## 🛠 Tech Stack

| | |
|---|---|
| Language | Swift |
| UI Framework | SwiftUI |
| Architecture | MVVM |
| Platform | iOS 16+ |
| IDE | Xcode |

---

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/kaekhara/AppNews.git
   ```
2. Open `AppNews.xcodeproj` in Xcode
3. Select a simulator (iPhone 14 or later recommended)
4. Press `Cmd + R` to run
